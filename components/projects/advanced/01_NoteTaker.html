Note Taker

The example is fairly complex and shows the implementation of a note taking application. It is basically one of the final projects as it involves a lot of programming concepts from JavaScript. It also stores the messages locally in the database of the browser.

Included programming concepts:
  • Arrow Functions
  • API Calls
  • Local Storage
  • Callbacks
  • Short JavaScript notions
  • Built-In Objects
  • Classes
  • Dom Elements

Features:
  • Create notes 📝
  • Light Mode ☀️ & Dark Mode 🌘
  • Store notes 💾
  • Edit Notes ✏️
  • Dbl Click to delte notes 🗑️
  • Date and time of creation (ordered by 📅
Info!

The programm is running inside an iframe window. The site itself is hosted somewhere else. The link to the website is https://notetaker-basic.netlify.app/.
